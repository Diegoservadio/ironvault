Architettura

Lo Studio Contabilità Milano

Infrastruttura ibrida on-premises + cloud che simula una PMI italiana reale

Windows Server 2022

Active Directory con dominio contabilita-milano.local, cartelle condivise, Group Policy NIS2, audit log Event Viewer.

PC Laura Bianchi

Windows 10 Education unito al dominio. Punto di ingresso simulato del ransomware via phishing email.

Ubuntu Server 22.04

File server con backup agent AWS CLI, cron job ogni ora, Grafana + Prometheus per il monitoraggio I/O.

S3 Object Lock WORM

Bucket con modalità COMPLIANCE attiva. Nessuno può cancellare i backup per 30 giorni — nemmeno con credenziali rubate.

IAM Least Privilege

Utente pmi-backup-agent con solo PutObject su S3. Il DeleteObject è esplicitamente negato dalla policy.

EC2 t3.micro Recovery

Istanza di emergenza in standby. Si avvia su comando, ripristina i dati da S3 e riporta l'azienda online.
