02 — La Soluzione

Architettura IronVault

Infrastruttura ibrida on-premises + AWS con backup immutabile e disaster recovery automatico. Costo ~15 euro/mese per una PMI reale.

ON-PREMISES — VMware
  • Ubuntu Server 22.04 — file server + backup agent + Grafana SOC
  • Windows Server 2022 — Active Directory + GPO IronVault
  • Windows 10 — postazione Laura Bianchi (punto attacco)
  • Apache2 — intranet aziendale Studio Contabilità Milano
  • Backup automatico ogni ora via cron su S3
AWS CLOUD — eu-north-1
  • S3 Object Lock COMPLIANCE 30gg — backup WORM inviolabile
  • IAM Least Privilege — DENY esplicito su DeleteObject
  • Bucket Policy — DENY assoluto anche per admin console
  • EC2 t3.micro standby — recovery in ~2 minuti
  • CloudWatch + SNS — alert accessi non autorizzati
Protezioni S3 implementate
🔒

Object Lock COMPLIANCE

Nessuno può cancellare i backup per 30 giorni — nemmeno root AWS

🚫

IAM DENY esplicito

pmi-backup-agent non può eseguire DeleteObject in nessun caso

🛡️

Bucket Policy DENY

Blocca il delete da qualsiasi Principal — incluso admin console AWS

📋

Versioning + Lifecycle

VersionId unico per ogni backup. Glacier dopo 30gg, eliminazione dopo 1 anno
